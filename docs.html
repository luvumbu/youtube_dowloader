YouTube Downloader

Application web locale + extension navigateur pour telecharger des videos YouTube.

PHP 8+ Python 3.x yt-dlp FFmpeg

▶ Installation automatique

Un seul script fait tout. Aucune connexion internet requise.

Double-cliquer sur install.bat a la racine du projet.

Le script fait automatiquement :

  1. Installe XAMPP (Apache + PHP)
  2. Installe Python
  3. Copie yt-dlp.exe dans C:\yt-tools\
  4. Extrait FFmpeg dans C:\yt-tools\ffmpeg\
  5. Copie le projet dans C:\xampp\htdocs\<nom_du_dossier>\ (detecte automatiquement)
  6. Cree les fichiers JSON necessaires
  7. Configure automatiquement Config.php
  8. Propose de lancer Apache et ouvre le navigateur
FichierProgrammeTaille

1 XAMPP

Uniquement si install.bat n'a pas ete utilise.

Hors ligne : program\xamp.exe | En ligne : apachefriends.org

Chemin : C:\xampp. Cocher Apache + PHP. Lancer Apache.

2 Python (optionnel)

Utile pour MAJ yt-dlp via pip. L'app fonctionne sans.

Hors ligne : program\python.msix | En ligne : python.org

3 yt-dlp

Telecharge les videos YouTube.

Hors ligne : copier program\yt-dlp.exe dans C:\yt-tools\

En ligne : pip install yt-dlp

4 FFmpeg

Convertit audio/video.

Hors ligne : dezipper program\ffmpeg.zip dans C:\yt-tools\ffmpeg\

En ligne : winget install Gyan.FFmpeg

5 Projet

Copier dans XAMPP.

Copier le dossier dans C:\xampp\htdocs\ (garder le nom du dossier tel quel). Creer downloads/ et data/ si manquants.

6 Configuration

Pas necessaire si install.bat a ete utilise.

Modifier 4 chemins dans classes\Config.php :

const YTDLP_PATH = 'C:\\yt-tools\\yt-dlp.exe';
const FFMPEG_PATH = 'C:\\yt-tools\\ffmpeg\\ffmpeg-8.1-essentials_build\\bin';
const PYTHON_PATH = 'C:\\Users\\TON_USER\\...\\python.exe';
const PHP_PATH = 'C:\\xampp\\php\\php.exe';

Doubler les antislash : C:\Users → C:\\Users

7 Verification

Lancer Apache → ouvrir http://localhost/<nom_du_dossier>/ (URL affichee par install.bat) → coller une URL → telecharger.

8 Demarrage auto (optionnel)

Raccourci xampp_start.exe dans le dossier Startup (Win+R → shell:startup).

⚛ Extension — Installation

L'extension ajoute un bouton de telechargement directement sur YouTube.

  1. Ouvrir brave://extensions/ (ou chrome://extensions/)
  2. Activer Mode developpeur (en haut a droite)
  3. Cliquer Charger l'extension non empaquetee
  4. Selectionner le dossier extension/ du projet

L'icone rouge apparait dans la barre du navigateur. Aller sur YouTube pour tester.

Mise a jour de l'extension

Apres modification des fichiers dans extension/, il faut recharger :

  1. Ouvrir chrome://extensions
  2. Cliquer le bouton refresh (fleche circulaire) sur l'extension
  3. Rafraichir la page YouTube ouverte

Les telechargements en cours ne sont pas interrompus. Le worker tourne sur le serveur. La file d'attente se restaure automatiquement.

▶ Extension — Utilisation

4 boutons flottants en bas a droite sur chaque video YouTube.

BoutonAction
DL rougeAjoute a la file d'attente avec tes preferences. Affiche 0%→100% puis vert ✓
Burger grisOuvre le panneau d'options : audio/video, format, qualite, couverture
Queue bleuOuvre la file d'attente : progression, boutons passer/relancer/reordonner
Cloche grisePanneau de notifications : fichiers ignores (doublons), erreurs, succes

Bouton DL inline

Un bouton rouge DL apparait aussi a cote des boutons Like/Dislike sur YouTube.

Anti-doublons

Si la video est deja dans la bibliotheque, le bouton est vert ✓. En cliquant, le panneau s'ouvre avec le message "deja dans ta bibliotheque" — tu peux forcer le re-telechargement via le panneau.

File d'attente

Tous les telechargements passent par la file d'attente (bouton bleu). Tu peux :

  • Passer un telechargement bloque (bouton ▶▶)
  • Relancer un telechargement en erreur (bouton ↻)
  • Reordonner les elements en attente (fleches ▲▼)
  • Supprimer un element (×)
  • Vider les termines sans toucher aux telechargements en cours

La file d'attente est sauvegardee et se restaure automatiquement en changeant de page.

Preferences

Les reglages (format, qualite) sont sauvegardes automatiquement. Le bouton DL utilise toujours les derniers reglages.

☰ Extension — Playlists

Telechargement de playlists entieres depuis YouTube.

Quand une playlist est detectee (sidebar ou page /playlist), un bandeau rouge apparait en haut :

  1. Le nombre de videos detectees s'affiche (se met a jour au scroll)
  2. Cocher Filtrer par duree pour definir min/max en minutes
  3. Le compteur se met a jour : "18 / 24 videos"
  4. Cliquer Tout telecharger
  5. Double barre de progression : video en cours (rouge) + total (vert)
  6. Les doublons sont automatiquement ignores et logges dans la cloche

★ Application web

Fonctionnalites de l'interface locale.

Onglets

Telecharger, Recherche, Bibliotheque, Profil. L'onglet actif est sauvegarde.

Bibliotheque

Gros boutons : Tout selectionner (bleu), Deselectionner (gris), Lire (rouge), Supprimer (rouge fonce). Drag & drop avec animation. Stats en haut (total, audio, video, disque, duree).

Theme

Bouton soleil/lune en haut a droite. Sombre = fond noir. Clair = style YouTube blanc.

Historique

Bibliotheque → bouton Historique. Vues, likes, annee pour chaque entree.

! Depannage

Erreur PHP

Chemins incorrects dans Config.php. Verifier avec where yt-dlp et where ffmpeg.

Telechargement ne demarre pas

Verifier PHP_PATH. Tester : C:\xampp\php\php.exe -v

yt-dlp erreur

Mettre a jour via le bouton dans la bibliotheque ou pip install -U yt-dlp

Apache port 80

netstat -ano | findstr :80 pour trouver le programme qui bloque.

Extension : serveur off

Apache n'est pas lance. Ouvrir XAMPP → Start Apache.

Extension : boutons n'apparaissent pas

brave://extensions/ → recharger l'extension. Si erreur JS, verifier la console (F12).

Video protegee

L'app retente 2 fois. Si echec, mettre a jour yt-dlp.
